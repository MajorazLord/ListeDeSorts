# ListeDeSorts
## Description du projet
Ce projet a été réalisé par Alexis DEPALLE.

Il consiste  en le développement d'une application mobile sous SailfishOS.

Ce projet peut se nommer ListeDeSorts et permet d'afficher les données d'un fichier xml de sorts pour JDR dans une listeView.

Un clic sur un élément permet de voir les informations du sorts dans une nouvelle page (nom, cible, distance, résumé, etc).

## Perspectives d'évolutions
Ce projet pourrait assez facilement recevoir de nouvelles fonctionnalités :
* Ajout et de suppressions de sorts
* Une sauvegarde XML
* Modification des données depuis la page de détail
